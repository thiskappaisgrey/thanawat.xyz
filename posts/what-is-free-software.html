What is Free Software

Quote

“Arguing that you don’t care about the right to privacy because you have nothing to hide is no different than saying you don’t care about free speech because you have nothing to say.” - Edward Snowden

The four freedoms

*What is free software? The four freedoms

  1. The freedom to run the program as you wish, for any purpose (freedom 0).
  2. The freedom to study how the program works, and change it so it does your computing as you wish (freedom 1). Access to the source code is a precondition for this.
  3. The freedom to redistribute copies so you can help others (freedom 2).
  4. The freedom to distribute copies of your modified versions to others (freedom 3). By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.

It’s free

Why should you use Free Software? It’s free

You don’t pay anything(but you should if you can) for it unless it’s hosted on a server.

Privacy

Privacy and Security

Free software, due to its nature of an open source code, is more transparent than proprietary software. Therefore, if it collects data on you, developers will know about it. (if the software is popular enough of course).

Security fixes can happen relatively quickly because of all the eyes looking at the code. You should care about privacy, even if you live in a “democratic” country! Authoritarians can and will get into power! It’s important that we limit the governments’ ability to infringe on the peoples’ rights

Longevity

Longevity

Because free software has its code open source, you can guarantee that your software will last for a very long time! For example, Emacs has lasted for over 40 years!

Hackability

Hackability Open source allows developers to troubleshoot the software that they use, fix those issues and contribute their fixes upstream! Developers can also add in their features easily.

Certain free software, like Linux and Emacs, allows for flexibility and customizability! So you can personalize your computer to your computer to your hearts content!
