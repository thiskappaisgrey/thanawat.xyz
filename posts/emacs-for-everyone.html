What is Emacs

  • Emacs is a “Lisp Interpreter”!
  • The main feature of Emacs is text-editing
  • People are able to write code to create packages, adding a lot of functionality to Emacs!
  • You don’t need to program to use Emacs

Why Emacs?

  1. Because Emacs is, in my opinion, a better system for writing than say, Microsoft Word
    • That is, if you know how to use it!
  2. Because it is free!

Editing Styles

There are two primary editing styles that allows you to write text on a computer faster. These are:

Vim

Vim focuses on Modal editing. It’s main philosophy is that most of the time when writing text, you are actually editing rather than writing. Vim makes editing a lot easier by splitting actions into different modes. These modes are:

  1. Normal - the mode for describing actions to text
  2. Insert - the mode for inserting text
  3. Visual - the mode for selecting text

Emacs

Emacs, the other hand, focuses on Macros, predefined actions that activates when you press a certain sequence of keys. For example C-n allows me to move down a line(without the arrow keys), and C-p allows me to move up one line. This is useful because you are in one state all the time and it doesn’t break your editing flow. Binding your Ctrl key to Caps-lock makes the Emacs bindings more ergonomic.
