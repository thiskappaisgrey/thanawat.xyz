The Vim Philosophy

Vim is a Modal text-editor! That means there are three different input modes that all does entirely different things! When you first enter Vim(or Vim’s emulations), you enter “Normal” mode. This scares people away because when they type, Vim does either nothing or weird things.

Vim’s Modes

Mode Description
Normal Mode This mode allows you to “describe” what you want to do to a piece of text
Insert Mode This mode allows you to create text!
Visual Mode This mode allows you to visually select text!

Demystifying Vim’s Normal Mode

Let’s connect the foreign concept of Normal Mode to a concept we all (should) know, Grammar! The two main parts of a complete sentence are:

  1. Action words - verbs. Examples:
    • delete
    • change
    • yank(means “copy”)
    • select
  2. Modifiers - adverbs
    • in
    • around
  3. Object - nouns. Examples:
    • word
    • paragraph
    • sentence

The basic scheme of a normal mode action is: Action word -> (sometimes) modifiers -> (optional) Number to repeat -> Text-Object.

For example, typing “dw” in normal mode literally translates to: “delete a word”. To delete five words, you can do: “d5w”. You can also do someting like “di(” to delete everything in parentheses.

SPECIAL NOTE: if you’re working with paragraphs and sentences, doing something like “ds” to delete a sentence won’t work! You have to do “dis”(or “das”) to delete everything in/around the sentence, or similarly “dap”(or “dip”) for paragraphs.

The other modes, Insert and Visual

These modes are pretty self-explanatory. Like I said, you probably want to be in Normal mode as much as possible(unless you’re creating new text).
